## `phaeton-core sdk:link TARGETSDKFOLDER`

Symlink specific SDK folder during development.

```
USAGE
  $ phaeton-core sdk:link TARGETSDKFOLDER

ARGUMENTS
  TARGETSDKFOLDER  The path to the phaeton SDK folder

EXAMPLE
  sdk:link /path/to/phaeton-sdk/sdk
```
